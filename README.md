# 个性化网站管理系统

## 问题说明
当前网站使用浏览器的 `localStorage` 存储数据，这意味着：
- 数据只保存在**当前设备的当前浏览器**中
- 电脑上添加的网站，手机上看不到
- 更换浏览器或清除浏览器数据后，数据会丢失

## 解决方案

### 方案1：使用云存储服务（推荐）
将数据存储改为使用云数据库，如：
- Firebase Firestore
- 阿里云数据库
- 腾讯云数据库

### 方案2：添加数据导入导出功能
我已为您添加了数据导入导出功能，可以通过以下步骤实现设备间数据同步：

1. 在电脑浏览器中打开网站
2. 进入「网站页面」
3. 点击「导出数据」按钮
4. 将导出的JSON文件保存到手机
5. 在手机浏览器中打开网站
6. 点击「导入数据」按钮
7. 选择保存的JSON文件

### 方案3：使用同一浏览器账号登录
如果您在电脑和手机上使用同一浏览器（如Chrome）并登录了相同的浏览器账号，且开启了同步功能，那么localStorage数据可能会自动同步。

### 普通用户
可以注册新账号，但只能查看网站，不能添加或删除。

### 添加网站
1. 使用管理员账号登录
2. 点击「+ 添加网址」按钮
3. 填写网址、描述和分类
4. 点击「确认添加」

### 删除网站
1. 鼠标悬停在网站卡片上
2. 点击右上角的删除按钮
3. 确认删除
